RADAR CONDUCTOR BOGOTÁ - ANDROID

Cómo usar en Android:
1. Descomprime este ZIP.
2. Sube la carpeta a Netlify, Vercel o cualquier hosting.
3. Abre el enlace en Google Chrome del celular Android.
4. Toca Instalar app o menú de Chrome ⋮ > Agregar a pantalla principal.
5. Permite ubicación cuando la app lo solicite.
6. Usa Iniciar trabajo / Finalizar trabajo y guarda ingresos/gastos.

Nota:
- El GPS requiere HTTPS. Por eso debe estar publicado en un hosting, no abierto como archivo local.
- Waze se abre como enlace externo/app si está instalada.
- El historial se guarda en el almacenamiento local del navegador/dispositivo.
